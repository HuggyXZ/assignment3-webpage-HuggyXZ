Welcome to Study Task Manager

Hello! This app helps you stay on top of your study tasks. Log in to the Tasks page to add, view, search, and manage your assignments all in one place.

About the App

This study task management website is designed to help students organise their academic responsibilities in one central place. This app provides a structured way to track what needs to be done and when.

Key features of this application:

  • Create new study tasks with a title, course, due date, and description
  • Update existing tasks when details change
  • Delete tasks that are no longer needed
  • View all current tasks in an organised table
  • Login system to keep tasks private to each user

About Me

My name is Huy Nguyen. I am currently a first year student studying Information Technology at Adelaide University.

I usually manage my study schedule using a combination of handwritten notes and reminders, but I found that this approach is inefficient and difficult to manage. This website is my attempt to build something better.
