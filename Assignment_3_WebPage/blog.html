Project Plan

This project follows a simple 3-week sprint plan based on Agile ideas. The work is broken into smaller parts and completed step by step over time.

Week Phase Planned Work
Week 1 Planning and Analysis Read the assignment, create the repository structure, and plan the pages and features.
Week 2 Design and Implementation Build the home page and task management page using HTML, CSS, and JavaScript.
Week 3 Testing and Maintenance Test features, fix errors, improve the pages, and complete the blog and ethics reflection.
